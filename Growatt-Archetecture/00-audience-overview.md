# Growatt 企业平台架构概览

## 摘要

Growatt 企业平台通过统一 API、云边协同通信体系以及标准化与兼容并行的设备接入能力，为第三方平台、生态合作伙伴和系统集成方提供一致的接入边界与可持续演进的架构基础。平台在北向开放接口、云边通信链路、南向协议抽象和设备建模语义之间形成了相对清晰的分层体系，可支持多系列设备的统一管理与后续扩展。

本文档从整体架构视角，对 Growatt 企业平台的核心能力边界、接口定位、通信路径、协议关系和设备模型进行说明，并以多个视角页面帮助不同角色快速定位所需信息。

## 适用对象

- 业务决策者与生态负责人
- 平台架构师与解决方案架构师
- API 集成人员与第三方平台团队
- 设备接入、协议适配与数据建模相关团队

## 核心关注点

- 平台整体分层与核心能力边界
- Legacy API 与 OpenAPI 的对外定位
- 云端、采集器与设备之间的通信关系
- 标准化接入路径与 legacy 兼容路径
- 电池与逆变器之间的建模关系
